# Requirements

Project defining requirements 

### Requirements
* Sensors
  * Motion (X, Y, Z, Pitch, Roll)
  * Microphone
  * Temperature
  * Humidity
  * Light
  * Buzzer
* Interface
  * USB-C
    * UART
    * Power input
  * External connector
    * I2C
    * UART 
    * GPIO
    * Power output
  * 2,4 GHz antenna
    * BLE or WIFI
* Low power
* USB-C powered
* Battery powered
* Charging
* As small footprint as possible
* Design for test
* Current consumption measurement
* Power button
* Aux button
* Reset button
* Component placement on one side only
* SMD components only
* 2-3 mounting holes
* Config EEPROM
